Detailed Description

template<typename ObjectType, typename ObjectTypeId, typename DescriptorClass>
class UnknownEngine::Core::AbstractObjectFactory< ObjectType, ObjectTypeId, DescriptorClass >

Base class for object factories creating objects of single base type according to the passed descriptor.

Template Parameters
ObjectTypeBase object type to create (e.g. Component )
ObjectTypeIdObject type identifier (e.g. ComponentType )
DescriptorClassObject descriptor type (e.g. ComponentDesc )
