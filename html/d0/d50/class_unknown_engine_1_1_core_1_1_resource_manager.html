Detailed Description

Resource manager class.

Concept

Resource manager is purposed to load/unload resources. It provides user code with access to the data providers that do the actual load/unload/store work.

Usage

When a user wants to load a resource it calls the createDataProvider() method of resource manager. Resource manager finds the factory suitable to create the specified data provider type and delegates the creation of data provider to it.

When the plugin supporting data loading is loaded it creates the data provider factories and registers them with a call to addDataProviderFactory()
