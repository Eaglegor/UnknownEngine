Detailed Description

Frame synchronization message.

This message is broadcasted every frame by MainLoop. It may be sent more than once per frame.

enum UnknownEngine::Core::UpdateFrameMessage::Stage

Updating stage.

To separate some actions from another you can tell engine to send this message on different update stages. For instance, you can update physics and logic on preprocessing stage and rendering - on processing stage.
